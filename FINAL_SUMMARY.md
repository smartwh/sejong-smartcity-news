## 🔧 15일 기간 변경 방법

### 코드 위치
HTML 파일을 열어서 `fifteenDaysAgo` 부분 찾기:

```javascript
const fifteenDaysAgo = new Date(now.getTime() - 15 * 24 * 60 * 60 * 1000);
                                                  ^^
                                                이 숫자를 변경
```

### 예시

| 기간 | 코드 |
|------|------|
| 7일 | `7 * 24 * 60 * 60 * 1000` |
| **15일** | `15 * 24 * 60 * 60 * 1000` |
| 30일 | `30 * 24 * 60 * 60 * 1000` |
| 60일 | `60 * 24 * 60 * 60 * 1000` |

## 🎓 기술 상세

### 15일 필터링 로직
```javascript
// 현재 시간 기준 15일 전 계산
const now = new Date();
const fifteenDaysAgo = new Date(now.getTime() - 15 * 24 * 60 * 60 * 1000);

// 각 뉴스의 발행일 확인
const newsDate = new Date(news.pubDate);

// 15일 범위 내의 뉴스만 포함
if (newsDate >= fifteenDaysAgo && newsDate <= now) {
    // 대시보드에 표시
}
```

### 시간 표시 순서
1. 방금 전 (1분 미만)
2. 3분 전 (1~60분)
3. 2시간 전 (60분~24시간)
4. 어제 (정확히 1일)
5. 5일 전 (2~7일)
6. 3월 30일 10:45 (7일 이상)
